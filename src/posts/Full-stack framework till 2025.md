# Full-stack framework till 2025
:svelte:

UI: Svelte with Typescript
Meta framework: SvelteKit (in beta) or Astro.build
CSS: Tailwind
CSS Component: DaisyUI
Backend: Firebase
Testing: Cypress, Playwright
Payment: Stripe
Search: Algolia
Transactional mail: SendGrid
