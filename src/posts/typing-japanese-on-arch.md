# Setup

Use `fcitx` and `mozx`.

Configure:
Trigger input method: `alt+lshift`
Scroll between input method: `ctrl_shift`

Your input method should contain:

- English
- Mozc
- Hangul
- Google pinyin

Use `alt+lshift` to trigger input method.
Then use `ctrl_shift` to cycle between mozc, hangul or pinyin.
